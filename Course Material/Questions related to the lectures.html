What are advantages and disadvantages of MESI in comparison with MSI?

ADV: less unnecessary communication over the bus

DIS: additional bus lines, more complex state machine, waiting for all processors to set the shared line before it is possible to change state.

 

What are the major features of MOESI protocols?

  • ADDS NOTION OF OWNERSHIP TO AVOID UPDATING MEMORY ON EVERY WRITE; OWNER EVENTUALLY UPDATES MEMORY
  • ALLOWS CACHE-TO-CACHE TRANSFERS BETWEEN OWNER AND REQUESTER
  • OWNERSHIP IS TRANSFERRED TO ANOTHER CACHE OR TO MEMORY WHEN BLOCK IS INVALIDATED/REPLACED

What are the information that have to be exchanged between two caches (L1 and L2) for the snooping protocol with multilevel caches? Assume that the caches are inclusive.  Describe information exchange both in L1->L2 and L2->L1 directions.

  • If the block is in modified state in L1 then it must be marked modified in L2 as well. Data can be transferred to L2 or we can add additional state to L2 which is called "modified but stale".
  • If the cache controller observes bus transaction (BusRdX) the invalidation must be propagated to the L1 cache if the data is present in it. To do that, L2 cache can keep track of all the copies that are in L1.
